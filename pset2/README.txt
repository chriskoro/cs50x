Further instructions for how to work with each program are at the top of each C file
Main files / programs: caesar.c, initials.c, vigenere.c


caesar.c - Uses caesar cipher to encrypt text given by user
by the key (number) given by user at command prompt
Usage: ./caesar key

initials.c - Prints user initials with user providing name 
Usage: ./initials

viginere.c - Uses the Vigenere cipher to encrypt a plaintext 
message given by the user in program with the keyword given 
by the user at the command prompt 
Usage: ./viginere alphabeticKey
