Further instructions for how to work with each program are at the top of each C file
Main files / programs: hello.c, greedy.c, mario.c, water.c

basics: 
to compile (for example): make hello (***don't need to add '.c'***)
to run (usage): ./hello


hello.c - prints hello world
Usage: ./hello

greedy.c - User enters amount of change in program, and program
calculates the least amount of change to be returned
(ie 4 coins for .41 or 41 cents) using greedy algorithm
Usage: ./greedy

mario.c - User enters height of mario pyramid during
runtime and program prints it of that size.
Usage. ./mario

water.c - User inputs shower length and program
prints water bottles used
Usage: ./water
